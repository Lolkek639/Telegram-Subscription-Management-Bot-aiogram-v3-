# 🤖 Telegram Subscription Management Bot (aiogram v3)

Этот Telegram-бот предназначен для автоматизации управления платным доступом к приватному каналу или группе (на примере проекта **«Кавказ Группа»**). Бот берет на себя рутинную работу по проверке чеков, одобрению заявок, отслеживанию сроков подписок и очистке канала от истекших участников.

## ✨ Основные функции

* **Автоматическое одобрение заявок:** Бот использует `chat_join_request`, сверяет ID пользователя с базой данных и мгновенно принимает его в канал, если подписка оплачена и активна.
* **Гибкая тарифная сетка:** Поддержка нескольких тарифов (например, доступ на 5 месяцев или вечный VIP-статус) с возможностью выбора валюты (Рубли / Тенге).
* **Менеджер подписок (APScheduler):** Фоновый планировщик каждый час проверяет базу данных и автоматически исключает пользователей, у которых истек срок действия тарифа, отправляя им уведомление.
* **Уведомления об истечении:** Автоматическая рассылка напоминаний пользователям за 3 дня и за 24 часа до окончания доступа.
* **Удобная Админ-панель:** * Интерактивная статистика (кол-во пользователей в базе и активных подписчиков).
  * Функция массовой рассылки сообщений всем пользователям бота.
  * Ручной бан/удаление пользователя из канала по его Telegram ID прямо через интерфейс бота.
  * Удобное одобрение или отклонение присланных скриншотов-чеков в один клик с помощью инлайн-кнопок.

## 🛠️ Технологический стек

* **Язык:** Python 3.10+
* **Фреймворк:** [aiogram v3](https://github.com/aiogram/aiogram) (Asynchronous Telegram Bot API)
* **Планировщик:** [APScheduler](https://github.com/agronholm/apscheduler) (AsyncIOScheduler)
* **База данных:** SQLite3
